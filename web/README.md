# Help Desk (Web)

Projeto final do curso Fullstack da Rocketseat, desenvolvido para fins de estudo.

## Visao geral
Aplicacao web do projeto Help Desk. 

## Stack
- React
- Vite
- TypeScript
- React Router
- Tailwind CSS
- Zod
- JWT (jsonwebtoken)

## Requisitos
- Node.js (LTS recomendado)
- npm

## Como rodar
1. `npm install`
2. `Copy-Item .env.example .env`
3. Ajuste `VITE_API_URL` no `.env`
4. `npm run dev`

## Scripts
- `npm run dev` - ambiente de desenvolvimento
- `npm run build` - build de producao
- `npm run preview` - preview do build

## Observacoes
Este repositorio foi criado para fins de estudo e e o projeto final para concluir o curso Fullstack da Rocketseat.
